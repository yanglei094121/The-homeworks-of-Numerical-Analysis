# 代码部分 python文件
定义了三个函数：

 1. v(x,n)函数：x可取任何值。当x为负值时，v(x,n)=1/v(-x,n);
 2. truev(x)函数：求$e^x$的真值；
 3. taylorv(x,n)函数：直接使用taylor展开求$e^{-2}$;
